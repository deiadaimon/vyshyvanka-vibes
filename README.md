# Vyshyvanka Vibes
 
Vyshyvanka Vibes is a high-performance landing page, fully adaptive across key breakpoints with high-density (**Retina**) image support.

Following strict **HTML5/CSS3** standards, the zero-debt codebase easily passes W3C validation, prioritizing **semantic markup** and ensuring maximum cross-browser stability via [modern-normalize](https://cdnjs.com/libraries/modern-normalize). By leveraging the **Concentric CSS** methodology and **CSS Custom Properties**, the project remains highly organized, scalable, and easy to theme.

The site is optimized for the modern web using **Vite**, ensuring peak performance, high **search engine visibility**, and full **screen-reader compatibility**.

## Tech Stack & Tools
- **Core:** HTML5 (semantic markup), CSS3 (modern layouts), JavaScript
- **Styling & Architecture:** CSS Variables for centralized theme management,
  Concentric CSS for property organization
- **Consistency:**
  [modern-normalize](https://cdnjs.com/libraries/modern-normalize) for
  cross-browser stability
- **Strategy:** Mobile-First Design, Adaptive Web Design (breakpoints: 375px,
  768px, 1280px, and 1440px)
- **Build System:** Vite
  ([Vanilla App Template](https://github.com/goitacademy/vanilla-app-template))
- **Optimization:** Image minification ([Squoosh](https://squoosh.app/)),
  high-density imagery Retina support, SEO best practices, Web Accessibility
- **Deployment & Validation:** GitHub Pages,
  [W3C Markup Validation Service](https://www.w3.org/developers/tools/)
